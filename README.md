# Docker_event
Dockerハンズオンイベント用リポジトリです。

## Setup✅
1. リポジトリのclone
```bash
git clone git@github.com:RareTECH-Event/Docker_event.git
```

2. .envを作成
```bash
cp .env.example .env
```
